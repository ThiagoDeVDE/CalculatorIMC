## Tecnologias
* Kotlin
* Activity to host the screens
* ViewGroup
    * RelativeLayout
    * LinearLayout
- Components de UI
    - TextView
    - EditText
    - Button
- Toast para display de mensagens
- Intent Explicita to open a second Activity
- FindViewById to recover UI components
- String resource to concatenate the text
